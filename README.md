# Stock Watcher（盯盘助手）

## 📋 系统要求
- Windows 7 及以上
- Python 3.8+ （如果从源码运行）

## 功能特点
- 极简悬浮窗：置顶显示，背景半透明，融入桌面。
- 全能监控：A股、港股、美股、黄金、外汇、数字货币全覆盖。
- 智能记忆：自动保存自选股列表及显示模式偏好。
- 便捷配置：内置常用指数与商品预设，一键添加。
- 配置简单：右键菜单设置添加/删除股票。
- 预警提醒：到达设定价格时抖动提醒，点击停止。

## 🚀 使用方法

### 方法1.直接运行（推荐）
下载Stock Watcher.exe文件直接运行

### 方法2.源码运行
**1.下载文件**
将以下文件放在同一目录：
- Stock Watcher.py
- requirements.txt
- install_and_run.bat

**2.初次运行**
双击 install_and_run.bat
脚本会自动：
- 检查 Python 环境
- 安装所有依赖
- 启动应用

**3.后续运行**
双击Stock Watcher.py即可

## 🔍资产代码说明
在 UI 设置面板中，必须先正确选择单选按钮（"股票/外汇" 或 "数字货币"），然后按以下规则输入代码：

### 1. 股票与指数 (选择 "股票/外汇")

**A股/国内指数 (经由腾讯 API)**
- 格式: 交易所小写拼音首字母 + 6位数字代码
- 示例: sh000001 (上证指数), sh600519 (贵州茅台), sz000858 (五粮液), sz399001 (深证成指)

**美股 (经由新浪 API)**
- 格式: gb_ + 小写股票代码
- 示例: gb_aapl (苹果), gb_tsla (特斯拉), gb_qqq (纳指ETF)

**港股 (经由腾讯 API)**
- 格式: hk + 数字代码
- 示例: hk00700 (腾讯控股), hk09988 (阿里巴巴)

### 2. 期货与外汇 (选择 "股票/外汇")

**国际大宗商品/外盘期货 (经由新浪 API)**
- 格式: hf_ + 品种大写代码
- 示例: hf_GC (COMEX黄金), hf_CL (WTI原油), hf_SI (白银)

**国内期货 (经由新浪 API)**
- 格式: nf_ + 品种代码
- 示例: nf_AU0 (沪金连续), nf_RB0 (螺纹钢连续)

**外汇汇率 (经由新浪 API)**
- 格式: 6位大写字母组合 (基础货币+计价货币)
- 示例: EURUSD (欧元/美元), USDJPY (美元/日元), USDCNH (美元/离岸人民币)

### 3. 数字货币 (选择 "数字货币")

代码内置了自动拼接 USDT 的逻辑，并提供币安 (Binance) -> 芝麻 (Gate.io) -> CoinGecko 的三级降级容灾。
- 格式: 代币大写简称 (最简写法) 或 完整交易对
- 示例:
BTC 或 BTCUSDT (比特币)
ETH 或 ETHUSDT (以太坊)
SOL (Solana)
DOGE (狗狗币)
