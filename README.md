# 盯盘助手

上班摸鱼神器：一个极简的、隐蔽的 Python 桌面盯盘小工具。

## 功能特点

*   **极简悬浮窗**：置顶显示，背景半透明，融入桌面。
*   **双模式显示**：
    *   **百分比模式**：极简文字，适合隐蔽摸鱼。
    *   **柱状图模式**：可视化红绿条，直观感受涨跌幅度（支持动态量程与圆角UI）。
    *   **价格开关**：右键可自由开启/关闭实时价格显示（3位小数）。
*   **智能逻辑**：
    *   **自动跨日重置**：每日自动清空历史波幅记录，精准反映当日振幅。
    *   **断点续传**：日内重启程序不丢失当日已发生的波动记录。
*   **实时更新**：对接腾讯/新浪财经双接口，支持股票、基金、期货、现货（金价）秒级刷新。
*   **全能监控**：
    *   **股市**：A股、港股、美股全覆盖。
    *   **贵金属**：独家支持国内现货金价（Au99.99）、国际伦敦金实时行情。
*   **技术面分析** (New)：
    *   **一键诊断**：右键个股即可查看 MACD、KDJ、RSI、均线、量比等多维技术指标。
    *   **综合研判**：基于多因子模型生成评分与操作建议（仅供娱乐）。
*   **智能记忆**：自动保存自选股列表及显示模式偏好。
*   **便捷配置**：内置常用指数与商品预设，一键添加。
*   **老板键**：双击隐藏/显示。
*   **配置简单**：右键菜单添加/删除股票。
*   **到价提醒**：盈亏转正或突破整数关口时抖动提醒。

## 🚀 使用方法

### 方法 1：直接运行（推荐）

1. **下载文件**
   - 将以下文件放在同一目录：
     - `stock_monitor_optimized.py`
     - `requirements.txt`
     - `install_and_run.bat`

2. **双击运行**
   - 双击 `install_and_run.bat`
   - 脚本会自动：
     - 检查 Python 环境
     - 安装所有依赖
     - 启动应用

### 方法 2：手动运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python stock_monitor_optimized.py
```

### 方法 3：打包为 EXE（无需 Python）

确保已安装 Python

双击 build_exe.bat

等待构建完成

在 dist 文件夹找到 Stock Monitor.exe
